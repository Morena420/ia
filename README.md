## Big Data – Las 5V

El **Big Data** se caracteriza por cinco propiedades principales conocidas como las **5V**:

- **Volumen:** gran cantidad de datos generados y almacenados.
- **Velocidad:** rapidez con la que se generan, procesan y analizan los datos.
- **Variedad:** diferentes tipos de datos (texto, imágenes, videos, sensores, etc.).
- **Veracidad:** calidad y confiabilidad de los datos.
- **Valor:** la utilidad o beneficio que se puede obtener al analizar los datos.
